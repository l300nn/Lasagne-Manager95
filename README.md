In diesem Spiel kannst du auf virtuelle Pferderennen wetten.
Es gibt verschiedene Pferde welche zufällige Farben und Quoten bekommen, dann kann man den Einsatz auswählen und die Wette setzen.
Das Game-UI ist an Arcade-Maschinen und alte MS-DOS Rechner inspiriert.
